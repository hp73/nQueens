Required to run: python 3.7


==============
nQueens.py
==============

Displays a brute force method and optimized method to solve the nQueens problem. The nQueens problem states that in a given chess board,
no two queens are allowed to threaten each other. This means that no queens may share the same row, column, or diagonal.
